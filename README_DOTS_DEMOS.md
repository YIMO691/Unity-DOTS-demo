# Unity DOTS Demo Project

This project has been prepared with beginner-friendly Unity DOTS demo scripts under:

`Assets/Scripts/DOTS/`

## First Unity Editor Steps

1. Open this project with Unity `2022.3.62f3c1`.
2. Wait for Package Manager to resolve the DOTS packages in `Packages/manifest.json`.
3. If Unity asks to restart or reimport, accept it.
4. Create one scene per demo or duplicate `Assets/Scenes/SampleScene.unity`.
5. Create a SubScene for each demo and put the demo authoring GameObject inside it.
6. Follow the README inside each demo folder.

## Demo Folders

- `Assets/Scripts/DOTS/Demo01_MovingCubes`
- `Assets/Scripts/DOTS/Demo02_BouncingBalls`
- `Assets/Scripts/DOTS/Demo03_FlockingAgents`
- `Assets/Scripts/DOTS/Demo04_TowerDefense`

## Notes

The scripts are ready, but prefabs, materials, SubScenes, and scene layout should be created from the Unity Editor so Unity can generate correct asset GUIDs and bake data.
